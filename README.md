# TAK Workspace

## Services

- WordPress: http://localhost:8080
- NCMAZ Frontend: http://localhost:3000
- Octarine Dashboard: http://localhost:5500
- OpenCore SaaS: https://localhost:4443
- phpMyAdmin: http://localhost:8081

## Start

```bash
# Copy and configure environment variables
cp .env.example .env
# Edit .env with your passwords and secrets
# Start services
docker-compose up -d
```
