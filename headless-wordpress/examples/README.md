# Examples Directory

This directory contains example implementations and integration code for your headless WordPress.

## Files

### `demo.html`

A standalone HTML demo page that demonstrates both GraphQL and REST API integration.

**Features:**

- Fetches posts using GraphQL
- Fetches posts using REST API
- Toggle between API methods
- Responsive card layout
- No build tools required

**To use:**
Simply open `demo.html` in your web browser while WordPress is running at `http://localhost:8080`.
